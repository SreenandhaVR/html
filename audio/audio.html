Audio Element

The video element can be used to play audio files,but the audio element is a bette choice.It's specifically designed for audio and provides clear semantic meaning making it more effective for both accesibility and search engines.Both elements share similar functionality so the syntax is pretty much the same with a few differences.

    <audio src="example-audio.wav" controls>
    <p> Your browser doesn't support HTML audio.
    Here is a <a href="example-audio.wav>link to the audio file</a>
    instead.<p>
    </audio>

Multiple Audio Files

    <audio controls>
    <source src="example-audio.mp3" type="audio/mpeg">
    <source src="example-audio.wav" type="audio/wav">
    <p>Your browser doesn't support HTML audio.Download the
    <a href="example-audio.mp3">MP3</a> file instead.

Supported Audio Files

  • MP3 - Mostly widely supported format.Compatible with all major browsers.Prvides good audio quality.
  • WAV - Offers high quality audio.Tends to have larger file sizes than MP3s.Is supported by ll modern browsers,but some legacy browers such as Internet Explorer may not.
  • OGG - An older open source option.Provides good audio quality.Smaller sizes
