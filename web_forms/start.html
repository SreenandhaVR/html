Web Forms

Web forms are one of the main ways users interact with websites. When we fill out a quiz, sign up for a newsletter, or enter payment details to make an online purchase, we are using forms to input data.

Forms can be basic and contain a couple fields like a name or email for newsletter subscriptions. They can also be complex, such as online shopping forms that require a mailing address, credit card number, billing address, and delivery address.

A form's complexity depends on how much information we need to be collected.

While most forms require processing by a web server through server-side programming, the user interface elements—the parts that we see in the browser, like text fields and submit buttons—are created with HTML.

On web development teams, front-end developers typically create the interface elements, while back-end developers manage the functionality and the data processing.

There are also many services for creating forms such as Typeform or Google Forms.

Form Controls

Elements that allow users to input data and interact with web forms (such as text fields, checkboxes, and buttons) are called form controls. Each form control has a specific role and purpose.

Form Syntax

Start with <form> tag and end with </form> tag and have two attributes:

  • action - where data is sent for processing; must be a relative or absolute URL.
  • method - how the form data is sent; GET or POST. 
<form action="" method="">

</form>

GET vs. POST

GET sends data as part of URL. Ideal for bookmarking and sharing.

Example:
https://store.com/search?q=black+pants

POST sends data in HTTP request body and keeps it hidden. It is more secure and ideal for sensitive information and large amounts of data.

The <input>Element

The <input> element is used for different types of data.

Include the name attribute to add a refernece to the data once the form is submitted

Example

  <form action=""" method="">
  <input name="email">
  <input name="password">
  </form>

The <label>Element

Used to create a caption for a form control

Example

  <form action=""" method="">
  <label>Email</label>
  <input name="email">
  <label>Password</label>
  <input name="password">
  </form>

The <button>Element

Sends the form data to the server for processing

The button text is added between the tags

Example

  <form action=""" method="">
  <label>Email</label>
  <input name="email">
  <label>Password</label>
  <input name="password">
  <button>Create account</button>
  </form>
