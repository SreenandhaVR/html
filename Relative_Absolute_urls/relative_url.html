Relative URLs

Relative URLs are used to link to resources that are located in the same directory or a subdirectory of the current page. They are specified relative to the location of the current page. For example, if you have an image file named "image.jpg" in the same directory as your HTML file, you can use a relative URL like this: . If the image is located in a subdirectory called "images", you would use: . Relative URLs are useful for maintaining links when moving files around within a website's directory structure. ../ is used to go back to the parent directory and .../ is used to go back to the grandparent directory. For example, if you have a file structure like this:

  • index.html
  • about.html
  • contact.html
  • images/
    • image1.jpg
    • image2.jpg

Then you can use relative URLs like this:

  • Home
  • About
  • Contact
  • Image 1
  • Image 2
