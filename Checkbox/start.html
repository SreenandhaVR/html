Checkboxes and Lists

Notes on HTML Checkbox and Select List Elements

Checkboxes and lists are HTML form controls used to collect user input. Checkboxes allow users to choose multiple options, while select lists let users choose one or more options from a drop-down menu.

The Checkbox Input

A checkbox allows users to select one or more choices from a list. Unlike radio buttons, multiple checkboxes can be selected at the same time.

  • Used to present one or more options.
  • Users can check or uncheck individual ptions
  • One or more options can be selected
  • Uses the <input type="checkbox"> element.
<input type="checkbox" id="html" name="skills" value="HTML">
<label for="html">HTML</label>

<input type="checkbox" id="css" name="skills" value="CSS">
<label for="css">CSS</label>

<input type="checkbox" id="js" name="skills" value="JavaScript">
<label for="js">JavaScript</label>
  • Uses type="checkbox".
  • Allows multiple selections.
  • The checked attribute selects a checkbox by default.
<input type="checkbox" checked>

The Radio Button

The type="radio" element creates a radio button.

The checked attribute ,selects a radio button by default.

  • Used for presenting multiple options where users can only select one choice.
  • Can only be deselected by choosing another radio button
  • Uses the <input type="radio"> element.
  • <input type="radio" id="radio"> <label for="radio">Option 1</label>

The Select List

The <select> element creates a drop-down list. Users can choose one option from the available choices. 

<label for="country">Country:</label>

<select id="country" name="country">
    <option value="india">India</option>
    <option value="usa">USA</option>
    <option value="uk">United Kingdom</option>
</select>
  • Uses the <select> element.
  • Each option is created using the <option> element.
  • The selected attribute makes an option selected by default.
<option value="india" selected>India</option>

The Multiple Select List

The multiple attribute allows users to select more than one option. 

<select name="languages" multiple>
    <option>HTML</option>
    <option>CSS</option>
    <option>JavaScript</option>
    <option>React</option>
</select>
  • Allows multiple selections.
  • Hold Ctrl (Windows) or Cmd (Mac) to select multiple options.
