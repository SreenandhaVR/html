Optimizing Images

Here are some tips for optimizing images on your website:

  • Use appropriate image formats (JPEG, PNG, SVG)
  • Compress images without losing quality
  • Use responsive images that adapt to different screen sizes
  • Implement lazy loading for images below the fold

Resolution

Resolution refers to the number of pixels in an image. Higher resolution images have more detail but also larger file sizes. It's important to find a balance between image quality and file size for optimal performance.

  • Width x Height; for example,1920 x 1080
  • Displays 1920 px horizontally and 1080 px vertically
  • Resolution is also independent of Physical size
  • Higher resolution = more pixels in the same space.

    • Pixel Density

    Pixel density refers to the number of pixels per inch or pixel per centimeter that fit into a screen

    Measures in pixel per inch(PPI)

    Higher pixel density = higher resolution images
