Types of Images

There are several types of images that can be used on the web, each with its own characteristics and use cases. The most common image formats include:

  • Vector graphics (SVG)

    • Images created with mathematical equations and shapes, which can be scaled and rotated without losing quality. Not suitable for photographic use.

  • Raster graphics (JPEG, PNG, GIF)

    • Images made from pixels (tiny squares of color that combine to form an image).The more pixels an image has, the higher its resolution and detail.

  • WebP

    • A modern image format that provides superior compression for web images, resulting in smaller file sizes without significant loss of quality.

  • HEIF/HEIC

    • A newer image format developed by Apple that offers better compression than JPEG, resulting in smaller file sizes with similar or better quality.

  • JPEG

    • displayes more than 16 million colors and is often used for photographic images.JPEG does not support transparency.

  • GIF (Graphic Interchange Format)

    • Only supports 256 colors. Works best for images with minimal color. Can be animated.Supports transparency

  • PNG (Portable Network Graphics)

    • Ideal for photos and images with color variations. Supports transparency. Supports animation. Supports 16 million colors.Supports lossless compression.File sizes vary based on type of image

  • WEBP

    • Developed by google. Supports photraphs, animation and transparency. Image Quality similar to JPEG and PNGs.Supports transparency. Compression is optimized for web, smaller file sizes.

  • SVG (Scalable Vector Graphics)

    • Vector image format. Can be scaled to any size without losing quality. Supports animation and interactivity.
