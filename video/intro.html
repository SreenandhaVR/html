Introduction to Video and Audio Elements

The introduction of the <video> and <audio> elements was a major milestone in HTML5. These elements made it easy to embed multimedia directly into web pages without relying on external plugins like Flash or Silverlight. They helped transform the web by enabling modern streaming platforms such as YouTube, Vimeo, and SoundCloud.

Supported Video Formats

  • MP4 – Supported by all major browsers and provides excellent quality with efficient compression.
  • WebM – An open-source format optimized for the web that delivers high-quality video.
  • Ogg – An older open-source video format that is less commonly used today but is still supported by some browsers.

Captions, Subtitles, and Chapters

HTML allows videos to include captions, subtitles, and chapter markers using the <track> element.

  • Captions display spoken dialogue and important sounds, making videos accessible to people who are deaf or hard of hearing.
  • Subtitles translate spoken dialogue into another language.
  • Chapters divide a video into sections so users can easily navigate to specific parts.

The <track> Element

The <track> element is a void element placed inside the <video> element. It is used to add captions, subtitles, descriptions, or chapter information to a video.

  • Use the src attribute to specify the location of the track file.
  • Track files use the WebVTT (.vtt) format.
  • The default attribute specifies the track that should be enabled automatically.

The kind Attribute

The kind attribute specifies the type of text track being added to the video.The srclang attribute must be included to specify the language.Uses the same language as the lang attribute.The srclang attribute is optional for other kind values. 

<track default src="captions.vtt" kind="captions" srclang="en">

Value Description
captions Displays captions for spoken dialogue and sound effects.
subtitles Provides translated dialogue for viewers.
chapters Defines navigation points within the video.
descriptions Provides text descriptions for visually impaired users.
metadata Contains additional information used by scripts.
