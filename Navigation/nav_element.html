The <nav> Element

The <nav> element represents a section of a page whose purpose is navigation.Represents a block of content that provides navigation links

Examples

  <nav>
  <ul>
  <a href="#home">Home</a>
  <a href="#about">About</a>
  <a href="#contact">Contact</a>
  </nav>
  </ul>
